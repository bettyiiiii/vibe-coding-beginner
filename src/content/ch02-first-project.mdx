# 用 AI 写出第一个作品

> 不用先学编程，跟着做，5 分钟后你就有第一个网页作品了。

---

## 什么是提示词（Prompt）？

**提示词 = 你给 AI 的指令。**

就像你教一个新同事做事，说的越清楚，他做得越好。AI 也是一样。

### 提示词的「三要素」

| 要素 | 说明 | 例子 |
|------|------|------|
| **角色** | 让 AI 扮演什么专家 | 「你是一个资深前端工程师」 |
| **任务** | 具体要做什么 | 「帮我写一个个人主页」 |
| **要求** | 约束条件和偏好 | 「用简洁的 HTML+CSS，中文内容，清新风格」 |

---

## 第一个提示词模版

```
你是一个资深前端工程师。

请帮我写一个【XXX】。

要求：
1. 使用【HTML/CSS/JS 或框架名】
2. 风格是【描述你想要的风格】
3. 内容是【描述内容】
4. 代码要有注释，方便我学习
```

<Callout type="tip" title="提示词技巧">
说清楚「你想要什么」比说清楚「你怎么想」更重要。

❌ 不好：「帮我写个网站」
✅ 很好：「帮我写一个个人作品集网页，深蓝色主题，有头像、简介、项目列表三个部分，用纯 HTML+CSS」
</Callout>

## 实战：写一个个人主页

### 第一步：写提示词

把下面这段直接复制到 Claude 或 ChatGPT：

```
你是一个资深前端工程师。

请帮我写一个个人主页，要求：
1. 纯 HTML + CSS，单文件，直接在浏览器打开就能看
2. 风格清新、专业，适合求职用
3. 包含：导航栏、个人介绍（头像占位+姓名+一句话标签）、技能列表、联系方式
4. 配色用深蓝色（#1a365d）+ 白色，字体用无衬线字体
5. 代码要加详细注释，帮我理解每一部分
6. 页面要适配手机屏幕

请直接输出完整 HTML 代码，我可以直接保存为 .html 文件使用。
```

### 第二步：把 AI 的输出保存为文件

1. 复制 AI 回复中的代码（从 `<!DOCTYPE html>` 到 `</html>`）
2. 打开电脑的「记事本」（Windows）或「文本编辑」（Mac）
3. 粘贴代码，保存为 `my-homepage.html`（注意扩展名是 `.html`）
4. 双击这个文件，它会在浏览器里打开！

### 第三步：看看效果 🎉

你的第一个网页作品已经诞生了！

<Callout type="info" title="遇到问题？">
- AI 输出的代码运行不了？把报错信息**直接复制**给 AI，它会帮你修。
- 想改颜色/文字？直接告诉 AI：「把背景色改成绿色，姓名改成我的名字」。
</Callout>

<AiExplainBtn explanation="这段代码的核心结构是：HTML 负责内容（标题、段落、图片），CSS 负责样式（颜色、大小、位置），它们通过 <style> 标签写在同一个文件里。浏览器读取后，按照 CSS 的规则把内容「画」出来。" />

## 实战：写一个待办列表网页

再来一个更有趣的例子——一个**可以真正使用的待办列表**！

### 提示词

```
你是一个资深前端工程师。

请帮我写一个待办列表（Todo List）网页，要求：
1. 纯 HTML + CSS + JavaScript，单文件
2. 功能：添加任务、标记完成（点击划线）、删除任务
3. 数据保存在浏览器的 localStorage，刷新页面不丢失
4. 界面简洁美观，绿色主题
5. 代码加详细注释，我是初学者，需要理解每一行
```

### 保存和运行

和上一个例子一样：复制代码 → 保存为 `todo.html` → 双击打开。

试着：
- 添加几个任务
- 点击任务标记完成
- 刷新页面，看看数据是否还在

<AiExplainBtn explanation="这个例子里的 JavaScript 代码负责「交互逻辑」：监听按钮点击、修改任务列表、更新页面显示。localStorage 是浏览器提供的「小笔记本」，可以帮你把数据存在本地。" />

## 小练习

尝试修改提示词，创造你自己的作品：

1. **改风格**：把个人主页改成「暗黑科技风」
2. **加功能**：给待办列表加上「筛选已完成/未完成」功能
3. **新作品**：让 AI 帮你写一个「个人简历网页」

<Callout type="warning" title="重要提醒">
不用纠结「代码对不对」——AI 写的代码，能跑就是对的。

你现在的重点是：**学会和 AI 对话，拿到能用的代码**。
理解代码是下一章的内容 😊
</Callout>
