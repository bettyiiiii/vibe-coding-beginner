### 2.3 实现「本地存储」

**提示词**：

```
【背景】这是我的代码：[粘贴当前完整代码]

【任务】添加 localStorage 持久化：
1. 页面加载时，从 localStorage 读取历史支出
2. 每次添加/删除支出后，自动保存到 localStorage
3. 刷新页面后数据不丢失

【要求】
- 使用 useEffect 实现（页面加载时读取）
- 使用 useEffect 监听 expenses 变化（变化时保存）
- 解释 useEffect 的依赖数组的作用
```

**关键概念讲解**（AI 会解释）：

```javascript
// useEffect：在组件渲染后执行副作用
useEffect(() => {
  // 页面首次加载时，从 localStorage 读取数据
  const saved = localStorage.getItem('expenses')
  if (saved) {
    setExpenses(JSON.parse(saved))
  }
}, [])  // ← 空数组：只在首次加载时执行

// 另一个 useEffect：当 expenses 变化时，自动保存
useEffect(() => {
  localStorage.setItem('expenses', JSON.stringify(expenses))
}, [expenses])  // ← [expenses]：只有当 expenses 变化时才执行
```

## 第五步：部署上线（让全世界看到）

### 方案 1：Vercel（推荐，免费）

1. 注册 [Vercel.com](https://vercel.com)（用 GitHub 登录）
2. 把代码推送到 GitHub
3. 在 Vercel 点击「New Project」，选择你的仓库
4. 点击「Deploy」—— 3 分钟后，你会得到一个 `https://你的项目名.vercel.app` 的网址！

### 方案 2：GitHub Pages（免费）

```bash
# 1. 安装 gh-pages
npm install -D gh-pages

# 2. 在 package.json 添加：
"homepage": "https://你的用户名.github.io/仓库名",
"scripts": {
  "deploy": "npm run build && gh-pages -d build"
}

# 3. 部署
npm run deploy
```
